\section{Introducción}

Este documento aborda el sistema de almacenamiento y recuperación de la información que ha sido desarrollado. En la sección \ref{decisiones} se tratarán las decisiones de diseño tomadas a la hora de desarrollar el sistema, así como el diseño de la base de datos y un diagrama de clases que detalla este sistema. En la sección \ref{manual} se explica brevemente el funcionamiento de esta aplicación, mientras que en la sección \ref{estadisticas} se comentan algunas estadísticas de funcionamiento del sistema. Para terminar, en el apéndice A se muestra el código del sistema completo.
