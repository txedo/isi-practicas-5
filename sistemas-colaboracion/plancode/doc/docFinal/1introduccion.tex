\section{Introducción}
% Propósito de la práctica para aplicar los conceptos de Groupware y CSCW vistos en teoría
\textbf{PlanCode} es un sistema \textit{groupware} cuyo objetivo principal es la elaboración de planes antes situaciones de emergencia de forma colaborativa. De este modo, este documento aborda el sistema \textit{groupware} que ha sido desarrollado. En la sección \ref{requisitos} se detalla la especificación de requisitos de dicho sistema; en la sección \ref{clasificacion} se realiza una clasificación de este sistema \textit{groupware}; en la sección \ref{requisitosFuncionales} se abordan los requisitos funcionales de un sistema \textit{groupware} que son satisfechos por esta herramienta; en la sección \ref{diseño} se plantea el diseño del sistema y, para terminar, se muestra en la sección \ref{manual} un pequeño manual de usuario de PlanCode.
