\section{Clasificación}

\subsection{Dimensiones espacio-temporal}
Atendiendo a la clasificación en tiempo y espacio propuesta por Johansen, \textit{Skype} puede clasificarse de la siguiente forma:

\begin{milista}
	\item \textbf{Mismo lugar, diferente tiempo}: aunque el propósito principal de \textit{Skype} es la comunicación por voz en tiempo real utilizando el protocolo VoIP (voz por IP), se puede clasificar en esta categoría porque también permite que los usuarios utilicen el chat que dicha herramienta incorpora, recibiendo los mensajes a posteriori si el usuario se encuentra desconectado. Es por ello que se trata, en esete caso particular, de una \textbf{interacción asíncrona}. ---- Realmente el chat es aqui o en \textbf{Diferente lugar, diferente tiempo}?? ------
	\item \textbf{Diferente lugar, mismo tiempo}: \textit{Skype} se puede clasificar dentro de esta categoría, ya que las llamadas de voz en tiempo real son síncronas, pero cada usuario puede encontrarse en diferentes lugares. Del mismo modo ocurre con el chat, cuando los usuarios participantes están conectados, ya que los mensajes enviados se envían a todos ellos.
\end{milista}

Por otra parte, atendiendo a la clasificación espacio-temporal propuesta por Grudin, la herramienta se puede clasificar en las siguientes categorías:

\begin{milista}
	\item \textbf{Mismo lugar, diferente tiempo (impredecible)}: ---- Como dice pizarra, ¿¿entraria aqui el chat, como en el caso anterior?? -----
	\item \textbf{Diferente lugar(impredecible), mismo tiempo}: --- Videoconferencia y llamadas
\end{milista}
